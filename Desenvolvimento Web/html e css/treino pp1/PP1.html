HTML e CSS

HTML é a única linguagem que o navegador consegue interpretar para a exibição de conteúdo. Este parágrafo foi escrito com fonte "Comic Sans MS", letras em tamanho 25pt na cor vermelha.

O HTML é um conjunto de tags responsáveis pela marcação do conteúdo de uma página no navegador. Diversas tags são disponibilizadas pela linguagem HTML e cada uma possui uma funcionalidade específica. Este parágrafo foi escrito com fonte "Arial Black", letras em tamanho 20pt na cor verde.

CSS é uma outra linguagem, separada do HTML, com objetivo de cuidar da estilização da página. A vantagem é que o CSS é bem mais rápido que o HTML para estilização. Com CSS, regras de estilo não aparecem mais no HTML. Este parágrafo foi escrito com fonte "Freestyle Script", letras em tamanho 40pt na cor branca.
